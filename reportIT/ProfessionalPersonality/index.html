  • D-支配性（Dominance）

D

考量一个人如何解决问题和应对挑战

强度：

D 值越高，在试图解决困难和障碍方面所表现出的积极性和进取心就越强。
D 值越低，则说明此人在做出决策前越倾向于多搜集信息。

当处于对抗性环境中时：

D 值高的人则会显得非常进取、积极。

情感：

D 因素用以考量一个人的易怒情感。
D 值极高者极易生气发怒；
D 值极低者不易生气发怒。
C

考量一个人对他人的期待和制定的规章制度如何做出反应

强度：

C 值越高，表明此人越能够遵守他人制定的规章制度。
C 值越低，表明此人更愿意挑战规章制度并寻求独立。

当处于对抗性环境中时：

C 值越高的人则更容易反应消极。

情感：

C 因素用以考量一个人的恐惧感；
C 值越高，说明此人的恐惧动因越强；
C 值越低，说明此人越无所畏惧。
I

考量一个人影响和说服他人的能力

强度：

I 值越高，在影响他人的思维方式方面，就越善于言辞，具有说服力。
I 值越低，越倾向于用数据和事实来说明问题。

当处于对抗性环境中时：

I 值越高的人反应越积极，且倾向于通过协商达成共识

情感：

I 因素用以考量一个人的乐观情感；
I 值极高者情绪非常乐观；
I 值低者则倾向于悲观。
S

考量一个人开展工作、承担责任的快慢

强度：

S 值越高，越倾向于一次有始有终地完成某件任务。同样，他们对变化有较强的抵触心理。
S 值越低，工作节奏越快，希望变化的愿望越强烈。

当处于对抗性环境中时：

S 值越高的人反应越消极。

情感：

S 用以考量一个人是否缺乏情绪；
S 值越高，越少情绪化，内心越难解读；
S 值越低，越情绪化，而且也越善于表达。
